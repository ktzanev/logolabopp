LaTeX

Pour utiliser les macros LaTex il faut inclure la librairie labopp (\usepackage{labopp}), qui est basée sur TikZ 3.x .

Si labopp.sty ne fait pas partie de votre distribution, vous pouvez le télécharger ici. Vous pouvez le mettre soit avec les autres bibliothèques LaTeX, soit dans le même répertoire que le fichier latex.

Attention :
  • Pour que l'en-tête et le pied de page se mettent en place, il faut compiler deux fois.
  • Si vous avez des problèmes avec la compilation (probablement dus à votre version de TikZ) :
    • Il existe une version de labopp.sty compatible avec l'ancienne version 2.1 de TikZ.
    • Il existe également une version simplifiée de la bibliothèque, laboppnotikz.sty, qui n'utilise pas TikZ. Pour pouvoir l'utiliser, en plus de la bibliothèque elle même, vous avez besoin de deux images en pdf : logonompp.pdf et labofooter.pdf. Pour l'utilisation de ce style vous pouvez consulter le modèle exemple_lettre_notikz.tex (ou sa version anglaise), ainsi que le résultat de la compilation avec pdflatex exemple_lettre_notikz.pdf.

  • Lettre type du labo n'utilisant pas TikZ

    Pour pouvoir utiliser ce modèle vous avez besoin de la bibliothèque, laboppnotikz.sty, et des deux images en pdf : logonompp.pdf et labofooter.pdf.
    • version française :
      • source LaTeX an ascii à compiler avec PDFLaTeX pour obtenir ce résultat en PDF;
      • source LaTeX en UTF8 à compiler avec PDFLaTeX, LuaLaTeX ou XeLaTeX (de préférence) pour obtenir ce résultat en PDF.
    • english version : LaTeX source that can be compiled with PDFLaTeX, LuaLaTeX ou XeLaTeX (best choice) to obtain this result in PDF.

  • Lettre type du labo utilisant TikZ

    Doit être compilée deux fois pour que l'en-tête et le bas-de-page se mettent en place. Utilise labopp.sty.
    • version française :
      • source LaTeX an ascii à compiler avec PDFLaTeX pour obtenir ce résultat en PDF;
      • source LaTeX en UTF8 à compiler avec PDFLaTeX, LuaLaTeX ou XeLaTeX (de préférence) pour obtenir ce résultat en PDF.
    • english version : LaTeX source that can be compiled with PDFLaTeX, LuaLaTeX ou XeLaTeX (best choice) to obtain this result in PDF.
